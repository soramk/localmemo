# WebMemoNote (Local Memo App)

**WebMemoNote** は、ブラウザ上で動作するローカルファイル直接編集型のプレミアムなMarkdown/テキストエディタです。
サーバーを介さず、PC上のローカルフォルダを直接マウント・編集できる「完全ローカルファースト」な設計が特徴です。

## 🎯 開発の背景と目的
本アプリは、**「仕事場や特定の環境においてアプリケーションのインストールが制限されているが、安全かつ快適にテキスト編集を行いたい」**というニーズに応えるために開発されました。

* **インストール不要**: ブラウザさえあれば、高度なエディタ環境を即座に構築できます。
* **高いセキュリティ**: 「File System Access API」を活用し、データはすべてあなたのPC内に完結します。外部サーバーへの送信は一切行われないため、機密性の高いメモも安心して編集できます。
* **ポータビリティ**: アプリ全体を1つのHTMLとして書き出す機能も備えており、オフライン環境やUSBメモリ等での持ち運びにも適しています。

## 🌟 主な機能

* **入力補完・入力支援 (New!)**:
  * **スラッシュコマンド (`/`)**: 日付挿入やMarkdownテンプレートを瞬時に呼び出し。
  * **内部リンク補完 (`[[`)**: プロジェクト内のファイルをサジェストし、簡単にリンクを作成。
  * **スマート・ブラケット**: 括弧や引用符の自動閉じ、選択範囲の囲み込み。
  * **動的提案**: ドキュメント内の既出単語から次の入力をサジェスト（設定でオフ可能）。
* **ローカルファイル直接アクセス**: サーバーを介さずPC上のフォルダを直接マウント。自動保存に対応。
* **高機能Markdownエディタ**: リアルタイム分割プレビュー、GitHub Flavored Markdown対応。
* **直感的なファイル管理**: ドラッグ＆ドロップによるフォルダ化、ネストされたツリー表示。
* **高度なカスタマイズ**:
  * **ダークモード** (ライト/ダーク/OS連動)
  * フォント、文字間、行間、配色の詳細設定。
* **画像ドラッグ＆ドロップ添付**: ローカルフォルダに自動保存し、リンクを挿入。
* **検索＆お気に入り**: スター機能による重要メモのピン留め。

## 🛠️ 技術スタック

* **Framework**: React 18 / Vite
* **Styling**: Vanilla CSS (モダンなデザインシステム、ダークモード完備)
* **Icons**: lucide-react
* **Markdown**: react-markdown, remark-gfm
* **Core API**: File System Access API, IndexedDB

## 🚀 使い方

### ローカルでの開発・起動
```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# プロダクションビルド
npm run build
```
