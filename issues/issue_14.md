# Issue: ファイル読み込み時の setIsPreviewMode エラー修正

## ステータス
**Closed (完了)**

## 概要
ファイルを選択して読み込もうとした際に、「`setIsPreviewMode is not defined`」というエラーが多発し、ファイルが開けない不具合が発生していました。

## 原因と対応
以前のアップデートで、エディタの表示モードを「編集 / プレビュー」の2択から「編集 / プレビュー / 左右分割(Split)」の3択へと進化させました。
その際、表示モードを管理する変数を `isPreviewMode` から `viewMode` という新しい仕組みへ移行しましたが、ファイルを開いた時の初期モードを設定する処理の中に古い `setIsPreviewMode` が1箇所だけ残ってしまっていたことが原因でした。

ファイル読み込み時の処理を以下のように修正しました。
* 古い処理: `setIsPreviewMode(true)`
* 新しい処理: `setViewMode('split')` （Markdownファイルを開いた時は、自動的に便利な2ペイン表示で起動するように設定）

## 完了状態
修正を適用し、エラーが発生せずにファイルが正常に開けること、およびMarkdownファイル読み込み時に自動で2ペイン表示になることを確認したため本Issueをクローズします。
