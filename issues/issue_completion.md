# Issue: 文字入力補完機能の追加

## 概要
ユーザーの入力効率を向上させるため、エディタに文字入力補完機能を追加します。

## 提案する機能
1.  **スラッシュコマンド (`/`)**:
    *   `/` を入力すると、頻繁に使用するアクションやテンプレートのリストを表示。
    *   例: `/today` (今日の日付), `/link` (Markdownリンクの挿入), `/table` (テーブルテンプレート)。
2.  **Markdown構文の補完**:
    *   記号を入力した際の自動閉じ（`[` -> `[]`, `(` -> `()`, `*` -> `**` など）。
3.  **ファイルリンク補完 (`[[`)**:
    *   `[[` を入力すると、現在のプロジェクト内のファイル一覧を表示し、選択するとリンクを挿入。
4.  **動的な単語提案**:
    *   入力中の単語に基づき、ドキュメント内で既出の単語を提案。

## UI/UX デザイン案
*   エディタ上のカーソル位置に追従するフローティングメニュー。
*   プレミアムな質感（背景のぼかし、洗練された影、Lucideアイコンの使用）。
*   キーボード操作（↑↓で選択、Enterで決定、Escでキャンセル）。

## 実装ステップ
1.  カーソル位置の座標取得ロジックの構築。
2.  補完メニューコンポーネントの作成。
3.  入力イベントの監視とトリガー条件の判定。
4.  各補完プロバイダー（ファイル名、コマンド、既出単語）の実装。
