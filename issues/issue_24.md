# Issue 24: セキュリティとプライバシーの可視化

## 概要
ユーザーがローカルメモを編集する際、データがインターネットに送信されていないか不安に感じるという懸念を解消するため、アプリの「ローカル完結性」を視覚的に保証する表示と詳細説明を追加する。

## 実施内容
- [x] サイドバーヘッダーに「ローカル専用 (Secure Local)」バッジを追加
- [x] ウェルカム画面にセキュリティ情報の要約セクションを追加
- [x] 「プライバシーと安全性」に関する詳細説明モーダルの実装
  - File System Access APIによる直接読み書きの説明
  - ネットワーク通信を行わないことの明記
  - ユーザー自身による確認方法（開発者ツール）の提示
