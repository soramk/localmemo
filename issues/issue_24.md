# Issue 24: セキュリティとプライバシーの可視化

## 概要
ユーザーがローカルメモを編集する際、データがインターネットに送信されていないか不安に感じるという懸念を解消するため、アプリの「ローカル完結性」を視覚的に保証する表示と詳細説明を追加する。

## 実施内容
- [x] サイドバーヘッダーに「ローカル専用 (Secure Local)」バッジを追加
- [x] ウェルカム画面にセキュリティ情報の要約セクションを追加
- [x] 「プライバシーと安全性」に関する詳細説明モーダルの実装
- [x] サイドバーの「新規メモ」ボタンと「設定」アイコンの高さを40pxに統一して整列
- [x] エディタ設定の大幅拡張
  - 数値入力とプリセット選択を統合したハイブリッドUIの導入
  - 設定の初期化時にデフォルト値を正しくマージするように修正（折り返し設定等の反映ミスを解消）
  - 右クリックメニューの構文エラーを修正し、機能が停止していた問題を解決
  - ファイル移動機能の強化（一階層上へ移動、指定フォルダへの移動、指定タブへの移動）
  - 多言語対応（日本語、英語、スペイン語）の導入。設定から切り替え可能
  - 設定画面のハイブリッド入力UIの視認性改善（アイコンとテキストの重なりを解消）
  - サイドバー空きスペースへのドロップで現在タブのルートへ移動
  - データを同じフォルダにドロップした際に削除されてしまう重大なバグを修正
